# dotfiles
macOS config dotfiles

## Committing Changes

First, ensure you're in the root folder `~/`. Make changes to a dotfile. Then, run the following commands:

```sh
    dotfiles <add/stage> <dotfile name>
```

```sh
    dotfiles commit -m <commit message>
```

```sh
    dotfiles push
```
